Cursor(커서) AI 에디터는 빠른 속도로 발전하고 있으며, 주요 버전 및 기능 업데이트 내역은 다음과 같습니다. 2026년 4월 기준, 최신 인터페이스를 통해 복수의 에이전트를 병렬로 실행하는 기능을 지원합니다. 

2026년 4월: 최신 인터페이스 업데이트. 로컬, 워크트리, 클라우드, 원격 SSH 환경에서 여러 에이전트를 병렬로 실행하는 기능 강화.
2025년 12월: Cursor 2.2 버전 업데이트. 비주얼 에디터 기능 추가, 컴포저(Composer) 기능 강화.
2025년 10월: Cursor 1.7 버전 출시. 프롬프트 입력 시 자동 제안 및 Tab 키를 통한 선택 기능 강화.
2025년 10월: Plan Mode(계획 모드) 도입. AI가 코드 작성 전 전체적인 계획을 세우고 단계별로 작업 수행.
2025년 중반: AI Rule 기능 (자주 사용하는 프롬프트 및 코드 컨벤션 미리 저장)


Claude Code(클로드 코드)는 2025년 5월에 정식 출시(또는 릴리즈)되었습니다.
2025년 (초기~중반): CLI 기반의 파일 분석, 편집, bash 명령 실행, Git 관리 기능으로 시작.
2025년 7월 (2.0~Beta): 안정성 향상 및 로그(history) 확인을 위한 claude-code-history-viewer 등 외부 툴 연동 시작.
2025년 10월 (2025 버전 종합): Subagents, Plan Mode, /rewind(작업 되돌리기), Plugin System, Skills, 터미널 UI 재설정 등 대규모 기능 추가.
2026년 1월 (2.1 버전): 커스텀 키바인딩, 새로운 태스크 관리 시스템, 히스토리 자동완성 등의 기능 추가 및 안정화 작업.
2026년 2월: 'redact-thinking-2026-02-12' 업데이트 이후 복잡한 엔지니어링 작업 실패 현상 보고 및 Extended Thinking 토큰 감축 (최대 73% 감소).

Google Antigravity (에디터/IDE)
Cursor나 Windsurf와 유사한 형태의 AI 네이티브 IDE입니다. 
2025년 11월 18일: 제미나이 3 출시와 함께 공개 프리뷰(Public Preview) 버전으로 발표.
주요 특징: 자율 AI 에이전트(Agentic AI)를 기반으로 하여 복잡한 코딩 작업 위임 가능.
초기 상태(2025년 11월 말): Gemini 3 Pro와 연동되어 출시되었으나, 초기 서비스 불안정 및 에이전트 종료 문제가 보고됨.
2026년 2월 이후: 맥, 윈도우, 리눅스 버전을 제공하며, Gemini 3 Pro의 하이/로우 모델 연동 및 나노바나(Nanobana) 이미지 생성 AI 연동이 강화되는 추세. 

OpenAI의 AI 코딩 에이전트인 Codex는 2025년 5월 16일(유튜브 영상 기준) 또는 그 직전인 5월 중순에 공개된 것으로 보입니다.
2025년 9월 16일: Codex-1 모델이 GPT-5-Codex 모델로 업그레이드되었습니다.
2025년 10월 5일: Codex 웹 버전 설정에 'Codex Alpha' 옵션이 추가되며 GPT-5-Codex의 업데이트가 예고되었습니다.
2025년 11월 13일: GPT-5.1 출시와 함께 새로운 Codex 버전 공개가 임박한 것이 확인되었습니다.
2025년 11월 17일: GPT-5.3-Codex가 출시되었습니다. 이 모델은 코드 생성, 디버깅, 리팩토링 능력이 대폭 향상된 가장 강력한 모델로 소개되었습니다.
2026년 2월 5일: GPT-5.3-Codex는 이전 버전보다 훨씬 빠르고 자원을 효율적으로 사용하며, 프로덕션 코드 처리에 적합한 일관되게 매우 좋은 성능을 보여주는 것으로 평가받았습니다

Pi 에이전트는 Mario Zechner가 개발한 미니멀리즘 철학의 코딩 에이전트로, OpenClaw(구 ClawdBot)의 기반이 되는 AI 도구입니다. 복잡한 기능을 최소화하고 핵심적인 4가지 도구(Read, Write, Edit, Run)를 기반으로 작동하는 것이 특징입니다. 
25년 12월 말 출시되었습니다.
Pi는 터미널 기반의 미니멀 코딩 에이전트로, 사용자의 워크플로우에 맞게 확장 가능한 구조를 제공
TypeScript 확장, 스킬, 프롬프트 템플릿, 테마를 조합해 자신만의 개발 환경을 구성하고, 이를 npm 또는 git 패키지로 공유 가능
15개 이상 AI 제공자와 수백 개 모델을 지원하며, 세션 중에도 모델을 전환하거나 사용자 정의 모델을 추가할 수 있음
세션은 트리 구조의 히스토리로 저장되어 이전 지점으로 이동하거나 HTML·GitHub gist로 내보내기 가능
핵심 기능을 최소화하고 확장성 중심의 설계 철학을 유지해, 개발자가 필요한 기능만 직접 구현하거나 패키지로 추가할 수 있음

오픈클로(OpenClaw)는 2026년 초부터 테크 커뮤니티와 뉴스에서 큰 주목을 받고 있는 AI 자율 에이전트 프로젝트로, 페터 슈타인베르거(Peter Steinberger)가 개발한 오픈 소스 기반의 툴입니다
오픈클로가 오케스트레이션을 하고 클로드코드나 코덱스등 cli 기반으로 일을 시키는 패턴도 사용 가능.
