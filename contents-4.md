Claude Code (Dev/Ops)
Stitch (Design) - https://stitch.withgoogle.com/
taskmaster (PM) - https://www.task-master.dev/

PRD 는 일반 채팅을 하며 deep research 나 notebooklm 을 활용하여 리서치하며 정리하고 
관련한 구체적인 TODO 와 진행상황 관리는 taskmaster 를 통해 작업을 분배하며 
frontend 등 디자인 시안은 stitch 를 통해 제작하고
전체 내용을 claude code 를 통해서 될때까지 무한 개발 진행한다 (랄프모드)
개발이 완료되면 자동으로 배포 (CI/CD) 하고
서버 구성과 운영은 그냥 서버에서 gemini-cli 로 바로 수행 해버린다.
